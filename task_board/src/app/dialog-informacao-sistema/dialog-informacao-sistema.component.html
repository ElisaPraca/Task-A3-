A primeira tabela lista as tasks cadastradas pelo usuário.

A segunda tabela exibe os registros criados através de uma fila. A terceira tabela exibe os registros criados através de uma pilha.

Nas tabelas 2 e 3 há um botão que remove o registro da respectiva tabela, sendo que na fila o primeiro registro a entrar é o primeiro a ser removido. Para a pilha, o primeiro registro é o último a ser removido.
